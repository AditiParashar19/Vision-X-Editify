## 🚀 About The Project

**Vision X Editify** is an AI-powered image editing and computer vision application designed to provide real-time image processing with an intuitive interface.  

It integrates powerful libraries like OpenCV and MediaPipe to deliver smart editing, detection, and visualization features — all inside a Streamlit web app.

---

## ✨ Features

✔️ AI-based Image Processing  
✔️ Real-time Image Preview  
✔️ Image Filters & Transformations  
✔️ Object Detection (Computer Vision)  
✔️ Clean and Interactive UI  
✔️ Fast and Lightweight
